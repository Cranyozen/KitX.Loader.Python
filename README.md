## 使用方法
> ❗ 您必须提前安装 Python3.8 及以上版本 及 KitX 的 dev=main 分支构建出的 KitX ❗
1. 下载或 clone 这个库
2. 打开 KitX Dashboard 并记录端口号 PORT
3. 使用命令 `python PythonLoader.py --load ExamplePlugin.py --connect 127.0.0.1:PORT`
> ❗ 您要手动替换 PORT 为实际端口号 ❗
